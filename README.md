# Seoul-Bike-Sharing-Demand
## 项目简介
本文针对Seoul Bike Sharing Demand数据集，采用回归分析、决策树回归、随机森林三种方法对数据进行拟合，探索影响共享单车需求的关键因素。通过对数据的分析，希望能够探究共享单车需求的变化规律，揭示影响共享单车需求的主要因素，并探索比较不同模型的拟合效果。
